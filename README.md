# hr_analysis

*description:-
This is my HR Analysis Dashboard built in Power BI to visualize employee data including attrition, age distribution, education, job satisfaction, and departmental trends. Provides actionable insights to help organizations understand workforce patterns and improve HR decision-making.

*Tech Stack Used:-
Power BI Desktop – Dashboard design & visualization
Power Query – Data cleaning and transformation
DAX – Calculations and measures (Attrition Rate, KPIs)
Excel / CSV – Data source

*Data Source:-
The dashboard uses an HR dataset stored in Excel, containing:
Employee ID, Age, Gender
Department & Job Role
Education Field & Level
Attrition status and job satisfaction

*Features & Highlights:-
Interactive Power BI dashboard
Employee attrition and demographic insights
Department, education, and job satisfaction analysis
KPI-driven design for quick executive reporting
